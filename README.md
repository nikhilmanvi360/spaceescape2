# 📖 Overview

Mission Override is an immersive browser-based coding game where players become the lead engineer aboard the fictional spacecraft **USCSS Aegis**. A catastrophic engineering failure threatens the ship, and players must restore its systems by solving programming and logical reasoning challenges before time runs out.

Instead of presenting questions as a traditional quiz, the application transforms each challenge into a mission inside a futuristic command interface.

The project was developed for a **college-level cluster technical event** to encourage students to learn programming through gamification.

---

# 🎯 Objectives

- Make coding practice engaging.
- Replace static quizzes with mission-based gameplay.
- Provide a futuristic user experience.
- Track player performance.
- Allow event organizers to monitor participants.

---

# 🧩 Story

The USCSS Aegis has suffered a major systems failure.

You are the chief systems engineer.

Core modules have failed, navigation is unstable, and reactor integrity is decreasing.

Every subsystem is protected behind engineering protocols.

To restore the ship you must solve coding puzzles, unlock secure modules, repair systems and successfully complete every mission before time expires.

Failure means total mission loss.

# 🎮 Gameplay

1. Login or enter the game.
2. Read the engineering directive.
3. Select a subsystem.
4. Solve programming or logic questions.
5. Submit the bypass code.
6. Unlock the next subsystem.
7. Continue until every system is repaired.
8. Receive mission statistics and score.
